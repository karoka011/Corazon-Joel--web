## Corazon Joel de Jesus - Proyecto "Arrullando a las Infancias"

Arrulladoras es un modelo de acompañamiento a niñas, niños y adolescentes hospitalizados, cuyo objetivo es reducir el estrés y la ansiedad que viven ellos y sus familias durante la hospitalización. Este sitio es la puerta de entrada del proceso de reclutamiento de voluntarios: comunica la misión del proyecto, captura el registro de aspirantes y sienta las bases para dar seguimiento ordenado a cada persona que quiere sumarse.

Zona piloto: Guadalajara, Jalisco.

## Tecnologia Involucrada
Fonrtend - HTML5, CSS3, JavaScript (vanilla, sin frameworks)
Base de datos(planeada)  - Firebase Firestore
Hosting - Firebase Hosting
Entorno de desarrollo  - Vs Code + Live Server Extension
